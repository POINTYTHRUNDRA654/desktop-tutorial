# Sim Settlements 2 Quick Start Guide

**Get up and running in 5 minutes**

## Installation (2 minutes)

### Easiest Path: Vortex Collection

1. Install **Vortex** from nexusmods.com
2. Click: **Sim Settlements 2 Starter** (or Starter+/Starter+ F4SE)
3. Let Vortex install everything
4. ✅ Done!

### Manual Path: Vortex

1. Install Vortex
2. Download and install these mods in order:
   - HUD Framework
   - Workshop Framework
   - Sim Settlements 2
   - (Optional) Chapter 2 & 3

3. Enable all in Mods tab
4. Make sure plugins are checked
5. ✅ Done!

### Mod Organizer 2

1. Install MO2
2. Download mods from Nexus (MOD MANAGER DOWNLOAD)
3. Right-click archives → Install
4. Enable in left panel
5. Check plugins tab
6. ✅ Done!

### Xbox

1. Launch Fallout 4
2. Mods → Search and Download:
   - Workshop Framework
   - Sim Settlements 2
   - HUD Framework
3. Load Order: Master files on top (auto-arranged)
4. ✅ Done!

---

## Starting the Game (3 minutes)

### Step 1: Activate Workshop
- Go to any settlement (e.g., Sanctuary)
- Activate the workshop terminal
- You now own the settlement

### Step 2: Build Beacon
- In workshop mode, build a **Recruitment Radio Beacon** (Miscellaneous → Power)
- Power it with generators
- Activate it

### Step 3: Wait
- Return to workshop
- The **Stranger** will appear shortly
- Talk to them to start the questline

### Step 4: Enjoy!
- Follow their quests
- Unlock new building plots
- Start managing settlements

## Essential Hotkeys & Tips

### Getting City Manager Holotape

| Method | Steps |
|--------|-------|
| **Quick** | Build City Planner's Desk → Craft → Holotapes → City Manager 2078 |
| **Vault 111** | Go to Overseer's Office → Terminal (Master lock) → Grab holotape |

### Holotape Main Functions

- **Manage Settlements**: Check plots, assign settlers
- **Toggle Quest Doors**: Fix location conflicts
- **Cheats**: Skip storyline if desired
- **Advanced**: Change main settlement, reboot quests

### Workshop Tips

| Tip | Benefit |
|-----|---------|
| Press **T** in workshop mode | Toggle build mode targets |
| Use **Place Everywhere mod** | Difficult plot placement |
| Pick up & cancel plot placement | Registers new location (after Place Everywhere) |
| Build Recruitment Beacon in ANY settlement | Questline triggers in any settlement with beacon |

## Verify Installation

✅ All of these should appear in-game:

| Check | Location | Requirement |
|-------|----------|-------------|
| Magazine | Sanctuary workshop | Read for building materials |
| Terminal | Vault 111 Overseer's Office | Master Lockpicking |
| Holotape | Same as terminal | Required tool |
| Stranger | Settlement with Beacon | Main quest NPC |
| HUD Elements | Workshop mode | Shows SS2 UI |
| SS2 Icon | Build mode | Confirms mod loaded |

If you don't see these:
- Check plugins.txt (make sure plugins are enabled)
- Verify load order matches above
- Restart game and mod manager

## First 30 Minutes

1. **Exit Vault 111** (0-5 min)
   - If new game, configure Prepopulation with City Manager Holotape
   - Or skip to step 2

2. **Go to Sanctuary** (5-10 min)
   - Activate workshop
   - Claim settlement

3. **Build Beacon** (10-15 min)
   - Place Recruitment Radio Beacon
   - Power it
   - Activate it

4. **Meet Stranger** (15-20 min)
   - Wait in settlement
   - Stranger appears
   - Start conversation
   - Accept main quest

5. **Explore** (20-30 min)
   - Get City Manager Holotape
   - Read included magazine for building materials
   - Start first quest objectives
   - Build first settlement plots

## Common Issues Quick Fix

| Issue | Fix |
|-------|-----|
| **Stranger won't appear** | Try Tools > Advanced > Reboot Quest in holotape |
| **No HUD elements** | Enable plugins in Plugins tab |
| **Can't place plots** | Use Place Everywhere mod |
| **Quest won't progress** | Check Known Issues page for conflicts |
| **Save corruption (Xbox)** | Revert to backup save (always make hard saves before updates) |
