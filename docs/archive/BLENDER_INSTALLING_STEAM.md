# Installing Blender from Steam

Steam is a software distribution platform supporting Linux, macOS, and Windows. Blender can be downloaded, installed, and auto-updated via Steam.

## Install via Steam
1. Download and install the Steam client for your OS (steampowered.com).
2. Launch Steam and log in (or create an account).
3. Go to the Store tab, search "Blender", and click the green Install button.
4. Blender appears in your Library; click to launch.
5. Optional: right-click in the library list and add a desktop shortcut.

## Benefits
- Automatic updates: Steam applies new versions in the background.
- Cross-platform: same method on Windows, macOS, and Linux.
- Organized library: Blender sits alongside other apps in your Steam library.

## File Association Note
On **Linux** and **Windows**, Steam does not automatically associate `.blend` files with Blender. To enable this:

- **Linux**: follow the registration steps in [BLENDER_INSTALLING_LINUX.md](BLENDER_INSTALLING_LINUX.md).
- **Windows**: follow the registration steps in [BLENDER_INSTALLING_WINDOWS.md](BLENDER_INSTALLING_WINDOWS.md), or run `blender.exe -r` from the Steam installation folder.

## Updating with Steam
Steam automatically downloads and installs Blender updates when available. No manual action needed.
