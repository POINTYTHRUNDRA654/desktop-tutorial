# Bodyslide Quick Start Guide

## 5-Minute Setup

### What You Need
1. Bodyslide program
2. CBBE body mod
3. Bodyslide preset
4. Outfit mod with Bodyslide support
5. (Optional) Bodyslide patch if outfit unsupported

**Download all from Nexus via Vortex**

### Steps
1. Open **Vortex** ‚Üí Tools ‚Üí **Bodyslide**
2. Select your **Preset** (dropdown 1)
3. Select your **Outfit** (dropdown 2) - or leave blank for all
4. Click **Batch Build**
5. Leave all checkboxes marked
6. Click **Build**
7. Wait for completion message
8. Done! ‚úì

### Result
In-game: Your outfit now has your custom body shape applied.

## Keyboard Shortcuts Cheat Sheet

### BodySlide
- **CTRL + Click Build** = Build to working directory
- **ALT + Click Build** = Delete target files
- **W** = Toggle wireframe | **T** = Toggle textures | **L** = Toggle lighting

### Outfit Studio (Most Used)
- **0-8** = Select brush | **S + Wheel** = Resize brush
- **CTRL + Z/Y** = Undo/Redo | **CTRL + A** = Clear mask | **CTRL + I** = Invert mask
- **X** = X-Mirror | **F** = Transform Tool | **Q** = Vertex Selection
- **Del** = Delete shape | **G** = Toggle visibility
