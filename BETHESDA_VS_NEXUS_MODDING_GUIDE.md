# Bethesda.net Mods vs Nexus Mods - Complete Comparison Guide

## Overview

When modding Fallout 4, you have two primary platforms to choose from: **Bethesda.net** (built into the game) and **Nexus Mods** (external website and mod manager). Each has distinct advantages, limitations, and use cases. This guide will help you understand which platform is right for your needs.

## Bethesda.net Mods

### ‚úÖ **Advantages**

1. **Built-In Integration**
   - No external tools required
   - Access mods directly from the game's main menu
   - One-click install and enable

2. **Cross-Platform Support**
   - Available on PC, Xbox One, Xbox Series X|S, PlayStation 4, PlayStation 5
   - Same mod can work on multiple platforms (if within size limits)

3. **Simplicity**
   - Perfect for beginners
   - No mod manager learning curve
   - Automatic updates when mod author releases new version

4. **Official Support**
   - Backed by Bethesda
   - Direct integration with game launcher
   - Less likely to break with game updates

5. **Console Accessibility**
   - **Only** option for console players
   - Brings modding to platforms that traditionally couldn't use mods

### ‚ùå **Limitations**

1. **No F4SE (Fallout 4 Script Extender)**
   - Cannot use advanced mods that require script extensions
   - Limits complex gameplay overhauls
   - No SKSE-dependent mods

2. **File Size Restrictions**
   - **Console:** 2GB per mod, varies by platform total
   - **PC:** 5GB per mod (still limiting for large texture packs)
   - **Total mod space:** Console limits are strict

3. **No External Assets**
   - Cannot reference external files
   - All assets must be packaged with the mod
   - Increases mod file sizes

4. **BA2 Archive Only**
   - Mods must be packaged as BA2 archives
   - Cannot use loose files
   - More complex for mod authors

5. **Limited Load Order Control**
   - Basic drag-and-drop only
   - No advanced sorting algorithms
   - Harder to resolve conflicts

6. **Content Restrictions**
   - No adult/mature content
   - No copyrighted material
   - Stricter moderation

7. **Poor Version Control**
   - Can't easily roll back to previous versions
   - Limited update history
   - No changelogs requirement

8. **Smaller Mod Selection**
   - Many popular mods not available
   - Authors often prioritize Nexus
   - Fewer total mods

### üéØ **Best For:**
- Console players (only option)
- PC beginners who want simplicity
- Players who want "vanilla+" experience
- Small to medium gameplay tweaks
- Cosmetic changes (armor, weapons, settlements)

### ‚ö†Ô∏è **Not Ideal For:**
- Complex overhaul mods
- Large texture/mesh replacers
- Mods requiring F4SE
- Players wanting maximum control

## Detailed Feature Comparison

### Installation Process

**Bethesda.net:**
```
1. Launch Fallout 4
2. Click "Mods" from main menu
3. Log in to Bethesda.net account
4. Browse or search for mod
5. Click "Download"
6. Enable mod from load order list
7. Play
```

**Nexus Mods (Vortex):**
```
1. Create Nexus account
2. Download and install Vortex
3. Configure Fallout 4 in Vortex
4. Browse Nexus website for mods
5. Click "Mod Manager Download"
6. Vortex installs automatically
7. Deploy mods in Vortex
8. Launch game via Vortex
```

**Nexus Mods (Manual):**
```
1. Download mod from Nexus
2. Extract archive
3. Copy files to Fallout 4/Data folder
4. Activate plugin in launcher or tools
5. Manage load order manually
```

### Load Order Management

**Bethesda.net:**
- Simple drag-and-drop interface
- Basic conflict warnings
- Limited sorting options
- No automated conflict resolution

**Nexus (with tools):**
- LOOT auto-sorting
- xEdit conflict detection
- Dependency graphs
- Rule-based override system
- Master file management

### File Size Reality

**Bethesda.net Limits:**
- Console: ~2GB per mod, ~10-150GB total (varies by platform)
- PC: ~5GB per mod
- **Real-world impact:**
  - High-res texture packs impossible
  - Large quest mods split into multiple files
  - ENB presets not viable

**Nexus:**
- No limits
- **Examples of large mods:**
  - Tales from the Commonwealth: ~2GB
  - Sim Settlements 2: ~1GB
  - 4K texture packs: 10-20GB each
  - Full overhaul collections: 100+ GB total

## Platform-Specific Considerations

### Console Players (Xbox/PlayStation)

**You MUST use Bethesda.net** - It's your only option.

**Tips for Console Modding:**
1. **Stay within limits**: Monitor your mod space carefully
2. **Read descriptions**: Console-specific notes matter
3. **Disable mods for achievements**: Or use achievement mods
4. **Regular clean slate**: Occasionally start fresh to avoid corruption
5. **Load order matters**: Heavy mods at bottom, patches at top

**Recommended Console Mod Types:**
- Quality of life improvements
- Weapon/armor additions
- Settlement enhancements
- Minor gameplay tweaks
- Bug fixes

**Avoid on Console:**
- Massive texture overhauls (size limits)
- Complex script-heavy mods (performance)
- Mods requiring external tools

### PC Players

**You can use BOTH platforms**, but here's the smart approach:

**Recommended Strategy:**
1. **Primary: Nexus Mods**
   - Use for main mod list
   - Install via Vortex or MO2
   - Leverage F4SE mods

2. **Supplement: Bethesda.net**
   - Only if mod isn't on Nexus
   - Simple mods that don't need mod manager
   - Console-exclusive ports

**Why prioritize Nexus on PC?**
- No artificial limitations
- Full mod manager support
- Better version control
- Larger mod selection
- Active community support

## Common Misconceptions

### ‚ùå **"Bethesda.net mods are safer"**
**Reality:** Both platforms moderate content. Nexus has virus scanning and community reporting. Safety depends on downloading from official sources.

### ‚ùå **"Nexus is only for advanced users"**
**Reality:** Vortex makes modding as easy as Bethesda.net, with more power when you need it.

### ‚ùå **"Console mods are limited versions of PC mods"**
**Reality:** Often true, but some mods are built specifically for consoles and work great within the platform's constraints.

### ‚ùå **"You need Premium Nexus account"**
**Reality:** Free accounts work fine. Premium just removes speed limits and ads.

### ‚ùå **"Bethesda.net mods don't conflict"**
**Reality:** Conflicts happen on both platforms. Load order management is crucial everywhere.

## Migration Between Platforms

### Moving FROM Bethesda.net TO Nexus (PC)

**Why you might migrate:**
- Hit file size limits
- Want F4SE mods
- Need better load order tools
- Desire larger mod selection

**Migration Steps:**
1. Document your current mod list
2. Install Vortex or MO2
3. Find Nexus equivalents (most exist)
4. Start fresh save (recommended)
5. Re-install mods via mod manager
6. Configure load order with LOOT

**Important:** You can't transfer saves with Bethesda.net-only mods to Nexus setups easily.

### Using BOTH Platforms (Not Recommended)

**Why it's problematic:**
- Conflicting load orders
- Duplicate installations
- Difficult troubleshooting
- Management complexity

**If you must:**
1. Use Bethesda.net for console-exclusive mods only
2. Keep lists separate
3. Document what comes from where
4. Disable one set before playing
